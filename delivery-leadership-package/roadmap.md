# Roadmap - Evergreen Quote Delivery

## Phase 1 (Current Project/Week)
 - Basic, responsive page is built with proper branding for Evergree
 - Users are able to get a quick quote from the basic calculator
 - Users only have to put in basic information to get quote
 - Testimonals are listed on the page

## Phase 2 (TBD)

 - Allow users to obtain more information about our services (About)
 - Allow returning customers to see their accepted coverage (Coverage)
 - Allow users to contact us if help is needed (Contact)
